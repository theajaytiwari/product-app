# 🛒 Product Management App

A full stack product management application built with React.js and Node.js.

## 🚀 Tech Stack
- **Frontend:** React.js, Material UI, React Router, Axios, CKEditor 5
- **Backend:** Node.js, Express.js, MongoDB Atlas, JWT, Helmet.js, Joi

## ✨ Features
- Login & Signup with JWT Authentication
- Product CRUD Operations (Add, Edit, Delete)
- Image Slider
- CKEditor Rich Text Editor
- Form Validation with Joi
- API Security with Helmet.js
- Responsive UI with Material UI

## ⚙️ How to Run

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev

## 🌐 Ports
- Frontend: http://localhost:5173
- Backend: http://localhost:5000
